
# Other Menu Options in vCenter

## Overview

Besides Hosts & Clusters and VMs & Templates, vCenter provides additional menu options for managing different parts of the infrastructure.

These sections help organize storage, networking, policies, and administration tasks.

---

## 1. Storage

Used to manage:

- Datastores (VMFS, NFS, vSAN)
- Storage policies
- Storage adapters
- Capacity monitoring

Administrators can view usage, create new datastores, and monitor performance.

---

## 2. Networking

Used to configure:

- Standard vSwitches
- Distributed Switches (VDS)
- Port groups
- VLAN settings

This section manages virtual networking across hosts and clusters.

---

## 3. Policies and Profiles

Includes:

- VM Storage Policies
- Host Profiles
- Customization Specifications

Used for standardization and automation across multiple hosts and VMs.

---

## 4. Administration

Central management area for:

- Roles and permissions
- Licensing
- Single Sign-On (SSO)
- Global settings
- Certificates

This section controls access and overall platform configuration.

---

## 5. Lifecycle Manager

Used for:

- ESXi patching and updates
- Upgrade management
- Baseline creation
- Host remediation

Ensures hosts stay updated and compliant.

---

## 6. Tasks and Events

Provides visibility into:

- Recent actions
- System warnings
- Error logs
- User activity

Helpful for troubleshooting and auditing.

---

## Summary

The additional vCenter menu options provide tools for managing storage, networking, security, updates, and system-wide settings. Together, they complete the centralized administration capabilities of the vSphere platform.
