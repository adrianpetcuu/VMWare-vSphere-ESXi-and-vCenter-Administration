
# vCenter Homepage Overview

## Overview

The vCenter Homepage (vSphere Client Home) is the main dashboard displayed after logging into vCenter.

It provides a centralized view of your entire virtual infrastructure.

Access via:

https://<vcenter-IP-or-FQDN>

---

## Main Sections

### 1. Hosts and Clusters
- View ESXi hosts
- Manage clusters
- Monitor virtual machines

### 2. VMs and Templates
- Create and manage VMs
- Deploy from templates
- Organize workloads

### 3. Storage
- View datastores
- Monitor capacity usage
- Manage storage policies

### 4. Networking
- Configure vSwitches
- Manage VLANs
- View distributed switches

### 5. Administration
- Manage users and roles
- Configure licensing
- Monitor tasks and events

---

## Global Inventory Tree

On the left side, you will see:

- vCenter Server
- Datacenters
- Clusters
- Hosts
- Virtual Machines

This tree structure allows easy navigation.

---

## Quick Actions

From the homepage, you can quickly:

- Create a new VM
- Add a host
- Create a cluster
- Enter maintenance mode
- Check system health

---

## Monitoring Panel

The homepage also shows:

- Recent tasks
- Active alarms
- Performance summaries
- Resource utilization

This helps administrators quickly identify issues.

---

## Summary

The vCenter Homepage is the central control panel of your VMware environment. It provides visibility, navigation, and quick access to all infrastructure components from a single interface.
