# VMware Horizon

## Overview

VMware Horizon is a Virtual Desktop Infrastructure (VDI) platform that delivers virtual desktops and applications to end users.

It allows users to securely access:

- Virtual desktops
- Remote applications
- Corporate environments

From:

- Laptops
- Thin clients
- Tablets
- Web browsers

---

## What Horizon Provides

VMware Horizon delivers:

- Centralized desktop management
- Secure remote access
- Virtual desktop pools
- Application publishing
- Remote work infrastructure

It is commonly used for enterprise desktop virtualization.

---

## Key Components

### 1. Horizon Connection Server

- Manages user authentication
- Assigns virtual desktops
- Handles connection brokering

---

### 2. Horizon Agent

Installed inside virtual desktops.

- Enables remote display
- Allows user sessions

---

### 3. Horizon Client

Installed on user devices.

- Connects users to virtual desktops
- Available for Windows, macOS, Linux, mobile

---

### 4. vCenter & ESXi

- Provide virtualization infrastructure
- Host virtual desktops

---

## Desktop Types

### 1. Full Clone Desktops
- Independent virtual machines
- Used for persistent desktops

### 2. Instant Clones
- Rapid deployment
- Efficient resource usage
- Ideal for non-persistent desktops

## Horizon vs Traditional PCs

| Traditional PC | VMware Horizon |
|----------------|----------------|
| Local OS and apps | Hosted in datacenter |
| Hard to manage at scale | Centralized control |
| Risk of data loss | Data stays in datacenter |
| Device-dependent | Accessible from anywhere |

---

## Summary

VMware Horizon is a virtual desktop solution that delivers secure, centralized desktops and applications from the datacenter to end users. It simplifies desktop management and enables secure remote access across devices.
