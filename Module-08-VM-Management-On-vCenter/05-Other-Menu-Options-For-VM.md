# Other Menu Options for Virtual Machines (VM)

## Overview

In vCenter, when you right-click a virtual machine, several additional menu options are available beyond basic Power On/Off.

These options allow administrators to manage, protect, migrate, and maintain VMs efficiently.

---

# 1. Power Options

- Power On
- Power Off
- Restart Guest OS
- Shut Down Guest OS
- Suspend

Used for controlling VM power state.

---

# 2. Snapshot Management

- Take Snapshot
- Manage Snapshots
- Revert to Snapshot
- Delete Snapshot

Used for temporary state capture before updates or changes.

---

# 3. Clone

- Clone to Virtual Machine
- Clone to Template

Used for duplication or standardized deployment.

---

# 4. Migrate

- Change compute resource (vMotion)
- Change storage (Storage vMotion)
- Change both compute and storage

Used for load balancing or maintenance.

---

# 5. Template Options

- Convert to Template
- Convert to Virtual Machine

Used for managing golden images.

---

# 6. Export

- Export OVF Template

Used to create portable VM copies.

---

# 7. Guest OS Options

- Install VMware Tools
- Upgrade VMware Tools
- Open Console

Used for OS-level integration and management.

---

# 8. Edit Settings

Allows modification of:

- CPU
- Memory
- Disk size
- Network adapter
- Boot options

Core configuration management.

---

# 9. Permissions

Assign user roles and access control at VM level.

---

# 10. Monitor

Access performance charts:

- CPU usage
- Memory consumption
- Disk I/O
- Network traffic
- Events and alarms

Used for troubleshooting and optimization.

---

# Summary

The VM right-click menu in vCenter provides full lifecycle management capabilities, including power control, cloning, migration, snapshots, export, monitoring, and configuration. Understanding these options is essential for effective VM administration.
