# Accessing vCenter Through HTML5 (Short Guide)

## Overview

The HTML5 vSphere Client is the main web interface used to manage vCenter and the entire VMware environment.

Modern versions of vSphere use only the HTML5 client (Flash is deprecated).

---

## How to Access

1. Open a web browser.
2. Navigate to:

   https://<vcenter-IP-or-FQDN>

3. Log in with:

   administrator@vsphere.local  
   (or another authorized account)

---

## What You Can Do

From the HTML5 interface, you can:

- Manage ESXi hosts
- Create and manage VMs
- Configure clusters (HA, DRS)
- Monitor performance
- Manage storage and networking
- Configure permissions

---

## Default Port

- HTTPS Port: 443

No additional software installation is required.

---

## Summary

The HTML5 vSphere Client is the primary web-based interface for managing vCenter. It provides full access to hosts, clusters, virtual machines, storage, networking, and administrative settings.
