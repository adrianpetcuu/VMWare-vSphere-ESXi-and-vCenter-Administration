# VMware Online Lab – Virtualization 101

## Introduction

VMware Online Lab – Virtualization 101 is a beginner-friendly hands-on environment designed to introduce the fundamentals of virtualization and VMware technologies.

It allows users to practice in a real VMware environment without installing anything locally.

---

## What Is a VMware Online Lab?

A VMware Online Lab is a cloud-based virtual environment where you can:

- Access VMware vSphere
- Interact with ESXi hosts
- Manage virtual machines
- Explore vCenter Server
- Practice basic administrative tasks

All through a web browser.

No physical hardware is required.

---

## Objectives of Virtualization 101 Lab

The main goals of this lab are:

- Understand virtualization fundamentals
- Learn how ESXi works
- Navigate the vSphere Client
- Create and manage virtual machines
- Explore basic networking and storage concepts

---

## What You Typically Practice

### 1. Accessing the Environment

- Log into vCenter Server
- Explore the inventory
- Identify ESXi hosts and datastores

---

### 2. Creating a Virtual Machine

- Select a host
- Choose compatibility and guest OS
- Allocate CPU and RAM
- Configure storage
- Attach network adapter

---

### 3. Managing Virtual Machines

- Power on and power off VMs
- Open the VM console
- Modify resources (CPU, memory)
- Take snapshots

---

### 4. Exploring Networking

- View virtual switches (vSwitch)
- Understand port groups
- Assign VLAN IDs
- Connect VMs to different networks

---

### 5. Exploring Storage

- View datastores
- Understand VM files (VMDK, configuration files)
- Monitor storage usage

---

## Skills Gained

After completing the lab, you should understand:

- How virtualization abstracts physical hardware
- The role of the hypervisor
- Basic VM lifecycle management
- Core components of vSphere
- Differences between physical and virtual environments
